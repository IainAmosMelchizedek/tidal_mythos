# Tidal Mythos

A single-file personal website for **Iain Amos Melchizedek**.

## Deploy on GitHub Pages

1. Create or open your GitHub repository.
2. Upload `tidal_mythos_final.html` or rename it to `index.html`.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/ (root)`
5. Save.

## Notes

- The visitor counter is a GitHub-safe static stub backed by `localStorage`.
- The generators are modular and local, with clear upgrade points for future model-backed systems.
- The file is self-contained for portability and simple editing.
